# E-commerce Management System
A comprehensive web application for managing categories and products with Excel file upload functionality. Built with React, TypeScript, and Tailwind CSS for a modern, responsive user experience.
# Core Functionality
- Category Management: Create, view, and manage product categories
- Product Management: View and edit products with inline editing
- Excel Import: Upload Excel files to bulk import products
- Data Validation: Comprehensive validation for all inputs
- Responsive Design: Works seamlessly on desktop, tablet, and mobile
# Technical Features
- Frontend-Only: No backend required - uses local storage for persistence
- Real-time Validation: Instant feedback on form inputs
- File Upload Security: Validates file types and handles errors gracefully
- Sample Data**: Pre-loaded with example categories and products
- Export Templates: Download sample Excel files for easy import
# Requirements
- Node.js (version 16 or higher)
- npm or yarn package manager
- Modern web browser (Chrome, Firefox, Safari, Edge)
# Installation & Setup

### Step 1: Clone or Download
# If using git
git clone <repository-url>
cd ecommerce-management-system
### Step 2: Install Dependencies
npm install
### Step 3: Start Development Server
npm run dev
### Step 4: Open in Browser
Navigate to `http://localhost:5173` in your web browser.
